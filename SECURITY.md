# Security policy

Read the complete [security and trust model](docs/SECURITY.md).

Please do not open a public issue for vulnerabilities involving credentials,
cross-profile access, request allowlist escape, registry-signature bypass, or
unintended remote mutations. Report them through this repository's private GitHub
security advisory flow.

Include the AgentWeb version, adapter and adapter version, a minimal reproduction,
and whether any real account state changed. Remove cookies, tokens, personal data,
addresses, payment details, and private URLs from the report.

The newest public release receives security fixes. Older development snapshots
may be asked to upgrade before a report is investigated.
